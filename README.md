## Cara Menggunakan Aplikasi

Anda dapat mengunduh dan menginstal aplikasi **Augmented Reality Pengenalan Hewan dalam Bahasa Inggris** pada perangkat Android.

### Langkah-langkah Penggunaan

1. Unduh dan instal file aplikasi (APK) pada perangkat Anda.
2. Unduh **marker** melalui tautan Google Drive yang telah disediakan.
3. Buka aplikasi, klik mulai, kemudian arahkan kamera smartphone ke marker yang telah diunduh atau dicetak.
4. Objek hewan 3D beserta informasi dalam bahasa Inggris akan muncul secara otomatis pada layar.

### Teknologi yang Digunakan

* **Unity 3D** sebagai game engine.
* **Vuforia Engine** untuk implementasi teknologi Augmented Reality (AR).
